Question-1: What are the different ways to select an element in the DOM?

  • 1.Using getElementById - Method--- document.getElementById("id-name");
  • 2.Using getElementsByClassName - Method--- document.getElementsByClassName(" class-name");
  • 3.Using getElementsByTagName - Method--- document.getElementsByTagName("any kind of tag");
  • 4.Using querySelector - specified CSS selector like (id, class names, types, attributes, values of attributes, etc) Method---
  • 5.Using querySelectorAll - Method--- document.querySelectorAll("p.intro");

Question-2: What is the difference between innerHTML, innerText, and textContent ?

InnerHTML, innerText, and textContent are the properties of the DOM (Document Object Model). They allow to read and update the content of HTML elements.

    innerHTML

  • Give output with all text and HTML tags inside an element.

    innerText

  • returns only text without html tags inside an element.

    textContent

  • returns all text, including hidden elements,inside an element.

Question-3: What is event delegation in the DOM?

Event delegation is a technique of js where we add evenListener of the parent element. By instead of attaching event listeners to multiple child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling directs an event to its target

  • Event start with child element and end with parent element.
  • It is a Default behavior of a browser.
  • Event bubbling are stoped by event.stopPropagation() method
